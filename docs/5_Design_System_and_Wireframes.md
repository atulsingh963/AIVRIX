# AIVRIX Design System & Wireframes

## 1. Grid & Layout System
*   **Max Width**: 1440px for content containers.
*   **Columns**: 12-column grid layout for desktop, 8 for tablet, 4 for mobile.
*   **Spacing Base**: 8px system (e.g., 8, 16, 24, 32, 48, 64, 96).

## 2. Core Components (Shadcn UI based)
*   **Buttons**:
    *   *Primary*: Electric Indigo background, white text, subtle glow effect, smooth scale on active state.
    *   *Secondary*: Outline with subtle background hover.
    *   *Ghost*: Transparent, subtle gray hover.
*   **Cards**: Glassmorphic effect with 1px `border-white/10`. Hover state elevates the card (translates Y up by 4px) and intensifies the border glow.
*   **Inputs**: Minimalist, dark background (`bg-zinc-900`), thin border, indigo focus ring.
*   **Navigation**: Sticky, translucent header with blur (`backdrop-blur-md`).

## 3. Wireframe Core User Flows
*   **Client Onboarding**: 
    1. Click "Book Demo" -> 2. Select Date/Time on Interactive Calendar -> 3. Fill requirement form (AI categorized) -> 4. Confirmation & Account Creation.
*   **Service Exploration**: 
    1. Landing Page -> 2. Services Mega-Menu -> 3. Specific Service Page (e.g., AI Chatbots) -> 4. Case Studies -> 5. Contact CTA.
